# Drupal/PHP
- Follow Drupal coding standards and PSR-12.
- Use dependency injection
- Avoid static service locator patterns unless Drupal conventions require it.
- Config changes must be exported and tracked.
- Modern OOP PHP: Interfaces, Traits, Service Objects, Plugins, Entities and Repositories, Unit/Function tests.
- Follow the coding style of those books:
  - [PHP the Right Way](https://phptherightway.com)
  - [PHP Brilliance](https://leanpub.com/phpbrilliance)
